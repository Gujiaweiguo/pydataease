PostgreSQL conversion decisions:
- SnowFlake-style IDs use BigInteger primary keys with autoincrement disabled on ORM models.
- Java/MySQL longtext-like payloads that carry structured data were promoted to JSONB.
- Epoch-millis timestamps remain BigInteger rather than DateTime.
- Boolean flags stay Boolean, including soft-delete and feature toggles.
- Alembic env reads database_url from app.settings.config.get_settings() and supports async/offline migrations.
- Baseline migration is usable without a running PostgreSQL instance via offline SQL generation.
